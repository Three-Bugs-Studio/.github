## About

Three Bugs Studio is an independent software studio founded by three developers passionate about building modern, scalable, and maintainable software.

We collaborate with startups, businesses, and creators to transform ideas into production-ready digital products through thoughtful engineering and clean architecture.

- RESTful APIs
